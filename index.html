FluoGen: An Open-Source Generative Foundation Model for Fluorescence Microscopy Image Enhancement and Analysis

FluoGen-Group
University of science and technology of China
2025
FluoGen Open Platform Code

We present FluoGen, a diffusion-based generative foundation model designed to improve the performance of AI-based methods in data-constrained fluorescence image processing.
